# C Programming for Beginners

This repository is a **beginner-friendly C programming teaching and learning resource**.
It is designed to help students understand **core programming concepts step by step**
using **simple examples**, **clear comments**, and **progressive lessons**.

The repository is structured **lesson-wise**, and each lesson focuses on a specific
concept in C programming, especially loops and matrix operations.

---

## Lessons Available

### Lesson 1: Sum of an Array using a `for` Loop
**File:** `1-calculates-the-sum-of-an-array-using-a-for-loop.c`

- Introduction to arrays
- Understanding the `for` loop
- Summing array elements
- Beginner-friendly comments
- Time and space complexity explanation

---

### Lesson 2: Matrix Sum using Nested `for` Loops
**File:** `2-Matrix-Sum-using-Nested-for-Loops.c`

- Introduction to 2D arrays (matrices)
- Nested `for` loops
- Matrix addition
- Printing (drawing) matrices
- Time and space complexity analysis

---

### Lesson 3: Matrix Multiplication using Nested `for` Loops
**File:** `3-Matrix-Multiplication-using-Nested-for-Loops.c`

- Rules of matrix multiplication
- Working with matrices of different sizes
- Triple nested loops
- Step-by-step multiplication logic
- Time and space complexity explanation

---

## Learning Goals

- Understand how loops work in C (`for`, `while`, `do-while`)
- Learn how nested loops are used for matrix operations
- Build strong foundations in algorithmic thinking
- Connect loop structures with **time complexity (Big-O)**
- Write clean, readable, and well-documented C programs
